# AlipayKit

An SDK kit for Alipay.

## Installation

Add `:alipay_kit` to the list of dependencies in `mix.exs`:

```elixir
def deps do
  [
    {:alipay_kit, <requirement>}
  ]
end
```

## Usage

For more information, see the [documentation](https://hexdocs.pm/alipay_kit).
